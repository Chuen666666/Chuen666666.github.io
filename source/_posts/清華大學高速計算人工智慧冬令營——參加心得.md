我與 [@閔捷](https://xieserver.com/) 一起報名參加了由清華大學叢集運算團隊所舉辦的 《清華大學高速計算人工智慧冬令營》（2025/1/29~31），我們被分配到了第四小隊，隊員除了 [@閔捷](https://xieserver.com/) 之外，還有 [@匿名用戶9487](https://qwo877.github.io/me/) 和 [@Xiancode207585](https://github.com/Xiancode207585)。我和 [@匿名用戶9487](https://qwo877.github.io/me/) 基本上算是舊識了，我們在 115 資訊月資訊應用技能競賽就見過了，[@Xiancode207585](https://github.com/Xiancode207585) 這個交大佬我們倒是都互不相識，不過合作和聊天過程意外地順利就是了。

## 第零天（28 號）

我和 [@閔捷](https://xieserver.com/) 因為聽說了清大提供住宿的環境不太好，因此我們提早自己訂了飯店，我和他就這麼住上了幾天，~~當初還不小心少訂一天，我們兩個北七甚至沒發現，出發前一天才去補訂~~。

[@閔捷](https://xieserver.com/) 訂了今天到新竹的台鐵，其實 12 點多的火車對於明天才開始活動的我們來說實在有些早，不過既然都訂了，我們就提早了不少出發。

到了新竹，我才猛然想起自己忘了帶小米手環的充電線，於是我們一路從火車站走去大遠百，去到裡面的小米實體店，我們目標明確，一進去就問了店員：「有沒有我這個手環的充電線？」但不幸的是，這麼舊型號的線已經只剩網購能買到了，我們也就只能作罷離開。我們隨後去查了查從大遠百到新舍商旅（即我們住的飯店）要怎麼去，結果發現從火車站過去還比較方便呢，但都走到這了，我們也不想花 100 多元搭 Uber，只好再花個半小時走過去，正好也就到了可以 Check-in 的時間。

原本我們是這麼想的，但結果走到剩 300 公尺時，有一段路被搭了，是施工，我們只好再拖著重重的行李繞路，最後花了超多時間和精力，總算是回到了飯店。

放好行李，並做了短暫的歇息後，我們開始找晚餐，然我我們得出了一個多數人都知道的結論——~~新竹根本美食沙漠~~（玩笑）。我們最後選擇了旅館樓下不遠的三媽臭臭鍋作為晚餐。

晚餐後的生活，大概這三天都差不多：跟女友語音通話，然後三個人（包含 [@閔捷](https://xieserver.com/)）嘴炮加幹話，再去買點酒、泡麵、零食，然後抱著筆電打 Code 到凌晨。

## 第一天（29 號）

今天是營隊正式開始的第一天，昨天和前台訂的 6:30 的 Morning Call 完全沒把我們叫醒，而是 [@閔捷](https://xieserver.com/) 在 8:30 左右驚醒，然後把我搖醒，我們再光速出門，騎 YouBike 一路狂飆到清大，~~還在旁邊的草叢那嘔吐了十分鐘~~，好不容易在 9:10 左右抵達，結果就得知了一個更震驚我們的消息——報到到 9:30，我們一邊~~罵街~~後悔，一邊暗暗決定後面些天要訂 Uber 來了。

這個營隊的上課模式和大學別無二致，基本上就是多數時候聽教授教，我們再在有實操的部分自己努力（~~或者煩隊輔~~）。今天的課程基本上就是教一些平行運算和超算的東西，以及一些叢集運算的規範。我們先在 iServer 註冊帳號，並利用 SSH 連到國網中心（國家高速網路與計算中心）的「創進一號」，我們會在上面跑一些練習或作業的 Code，來讓我們能夠了解何謂「叢集運算（HPC）」。

{% note info %}
創進一號（Forerunner 1, F1）為國網中心建置的高速計算主機，具備 558 台計算伺服器，共可提供 62,496 個計算核心；每台計算伺服器配備兩顆 56 核心 Intel Xeon Platinum 8480 + 2.0 GHz 處理器、InfiniBand 200 Gb/s 高速網路、512 GB 記憶體。

（資料來源：[國家高速網路與計算中心](https://www.nchc.org.tw/Page?itemid=2&mid=4)）
{% endnote %}

午餐其實並不是這次營隊包辦的項目，因此我們一行人得自己出去校外吃，我們最開始是決定和隊輔一同出去吃飯，不過隊輔也有和他同學約好，最後我們竟然高達了 21 個人一起前去吃午飯。我們去了間叫「紅吱吱」的平價牛排，雖算不上好吃，但第一個和所有隊友坐同一桌一同吃飯的一餐還是挺值得紀念的吧。

至於晚餐，營隊在第一個晚上提供了 Pizza（~~好耶又省一餐錢~~），因此我們是直接在教室吃，省時又好吃，是我今天吃得最滿意的一餐了。

但課程還遠沒有結束，晚餐過後還有課程，課程表定到 21:30 才結束，而實際上我們甚至是到了 22:30 多才離開學校的，除了課堂上的練習外，我們還有 Homework 要做，我們可以去拼 Scoreboard 排名，盡可能把程式改得快到極致（不過排名還是會被前面的一群又肝又佬的電神霸占），我們小隊哪懂 C++ 啊，於是一群 Vibe Coding 之神只好一直在換 AI 和提交屎山到 OJ 上之間反覆橫跳。

課程結束後，我們拖著疲憊的身體，原本想的是可以訂 Uber 回去，但又想到來時的上坡路要是不滑回去豈不是虧大了？於是我們又頭鐵地去借了 YouBike，一路滑下坡，吹著早上用體力和毅力換來的晚風，回到了房間。

但晚上除了幹話和我喝冰結 [@閔捷](https://xieserver.com/) 吃芭樂乾之外，我們還得用肝來換 Scoreboard 的排名，我「早早」地三點就睡了，但 [@閔捷](https://xieserver.com/) 獨自幹到了四點~~還不給我關燈~~，實在是很有毅力了。

## 第二天（30 號）

今天早上我們總算是起床得比較從容了，我們也學乖了訂 Uber 前往清大，不過今天因為要去參觀國網中心，我們得提早半小時前去清大，再坐遊覽車過去國網中心。

我們這幾個同組的人被分配到了相鄰的座位，不過在我旁邊的是 [@匿名用戶9487](https://qwo877.github.io/me/)，我們也藉這個機會好好地~~嘴炮~~交流了一番，也算是一個不錯的回憶了。

國網中心的參觀基本上都是聽演講，這對睡眠不足的我和 [@閔捷](https://xieserver.com/) 而言，要不睡著簡直是不可能的任務，但好在我們也沒有睡多久，最重要的是，聽到了 HiPAC 國網盃的比賽資訊，我們眼睛一亮，當即決定了要去參加這個比賽，更大的影響是，我特選正取的中教大（台中教育大學）原本我對它的資工還存疑，但隨著講者提到中教大的超算很強，這讓我對中教大完全改了觀。

國網中心的行程結束後，我們去成功湖拍了張大合照（下圖）就去吃午餐了。這次倒是沒有二十幾人的飯局了，反而只有寥寥幾人去，不過我、[@閔捷](https://xieserver.com/)、[@匿名用戶9487](https://qwo877.github.io/me/)、隊輔都在，還有一些隊輔的朋友，我們六人一桌，去吃了這幾天來最好吃的餐廳「甘泉魚麵」，雖然剛開始吃飯時有點小尬尷，但到後面也算聊起來了，社交的部分還算順利。

{% img /img/HPC_camp_group_pic.JPG 成功湖大合照 %}

下午和晚上的課程也是照例進行，主要在教 CPU、Core、MPI、OpenMP 等等，不知道是因為上手了，或者真的比較簡單，今天的實操體感上比昨天的順利不少，雖然仍然是需要依賴 AI，不過其實總體掌握感多過了昨天那種迷茫的感覺。

下午我們還有一個環節是參觀清大的圖書館，雖然去過了國家圖書館，已經看過了很厲害的圖書館，但作為校園的一部分，清大圖書館還是看得我相當震驚，領我們去看的領隊也耐心地向我們講解了一些關於清大的歷史和特色。

晚餐時間，今天清大依舊供應了一餐麥當勞，但我本人是對台灣老麥不太存有好感就是了。吃飯過程還有一個介紹清大資工系的節目，雖然特選結果都已塵埃落定，但我還是把它給聽完了，~~反觀 [@閔捷](https://xieserver.com/) 自己偷偷溜回教室卷 Homework 的排名~~。

晚餐和晚上的課程結束後，我們還有一個參觀清大機房的節目，這完全進入了 [@閔捷](https://xieserver.com/) 的舒適圈，但對硬體沒什麼研究的我，一趟下來只聽懂了機房有 5090 和 H100。不得不說，[@閔捷](https://xieserver.com/) 在這裡完全展現了自己對硬體的精熟和勇往直前的社交能力，他到處給別人講解機房中的機器，還主動和一堆清大生搭話，甚至一度被別人誤認成清大生，我則只能在旁邊聽著一堆專有名詞發呆。不過不得不說，機房是真的蠻大聲的，下面這張是我在機房裡測的分貝數，實在很難想像長期待在這種環境的人耳朵能變成什麼樣子。

{% img /img/CHUH_computer_room_sound.png 清大機房的分貝數 %}

今天放人的時間又是 22:00 多了，我們依然騎 YouBike 滑回飯店，但在某個紅燈時，我們發現了一個意外：「你外套呢？」我問道，「啊！忘在清大了，笑死。」[@閔捷](https://xieserver.com/) 回，這聽起來好像沒什麼問題，實則問題大了，明天就是退房日，但他將房卡放在了外套口袋中……。

晚上的生活誠如前面說的，就是嘴炮、打 Code 做作業、喝酒吃零食，依然是 3:00 睡覺。
