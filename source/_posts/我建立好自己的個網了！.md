## 起點動機
我之所以想架這個網站，除了因為大家都有，覺得很酷之外，也是因為在高中的時期沒有個網，想要整理做過的事情和製作備審，都變得困難和麻煩許多，因此想藉由這個網站來記錄我高中學測後的生活（包含三下、大學、研究所、出社會後）
## 食用方式
這個網站該如何使用應該一看就會了，所以這裡說點注意事項
1. 我很會~~尬聊~~講屁話，所以若想找我，都歡迎以留言或[關於我](https://chuen666666.github.io/about/)頁面的聯絡方式來找我
2. 我會盡量嚴格進行分類，所以文章應該很好找吧？
3. 想加入友情連結者，請至[友情連結](https://chuen666666.github.io/links/)頁面查看添加方式，或者直接與我聯絡也可以
4. 若想加入我的 Discord 伺服器，請私訊我；若想要加我的遊戲好友，也請至我的 Discord 個人介紹中加入（或直接私訊我）
5. 我分享的目的除了個人紀錄外，也只想順便分享經驗，我沒有特別強，因此不喜勿噴
6. 這個個網的 GitHub repo 是先將原 hexo 碼放上去，再用 GitHub Actions 自動 compile 並部署到 GitHub Pages 中，因此你完全可以從 repo 中看到我的原始碼
7. 有些東西我不一定會分享出來（例如架這個站的經驗與做法等），因此若想要敲碗某個主題，歡迎留言或私訊我
8. 很多東西我也會以影片的形式分享在我的 [YouTube 頻道](https://www.youtube.com/@chuen666) 中，因此也可以去那邊看看我的影片
9. 我真的沒有 IG 和 FB，所以不用再問了
10. 感謝大家來看我的個網，祝大家有個愉快的一天！
## 小小提醒
如果你也想架自己的個網，我建議直接叫 Gemini 教你就好，依我的經驗，它在架站上比 ChatGPT 強一些（甚至是免費版 Gemini 都比付費版 ChatGPT 強），所以可以直接跟著 AI 架即可（其他 AI 我是沒試過，試過的可以下面留言討論看看）
